# **Moonstruck Firebase Functions**

Moonstruck Firebase Functions는 moonstruck 서비스의 **백엔드 로직**을 처리하기 위해 서버리스 환경에서 동작하는 Firebase Functions 프로젝트입니다.

---

## **기술 스택**

- Firebase Functions
- TypeScript
- Google Vertex AI
- ESLint
